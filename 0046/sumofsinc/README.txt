sumofsinc.f90 is a slightly modified version of the Fortran code of https://fortran-lang.discourse.group/t/simple-summation-8x-slower-than-in-julia/1171 (certik's code).  The modification sumofsinc.f90 computes sum of sin(i)/i instead of sin(i).  Then the double of the sum plus one is equal to pi = 3.14159...
